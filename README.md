## Visionhub

Visionhub is a sleek and modern video streaming platform, built using the MERN (MongoDB, Express, React, Node.js) stack. It is designed to provide a seamless and engaging user experience, offering features inspired by YouTube, such as video playback, genre-based content exploration, personalized video suggestions, and robust search functionality. This application utilizes the YouTube API via RapidAPI to fetch and display video content dynamically.

## Features

### 1. **Video Genres**

- Browse videos across multiple genres, allowing users to explore content tailored to their interests.
- Dynamic loading of genre-specific videos for a personalized experience.

### 2. **Video Playback**

- Enjoy smooth video playback with essential controls (play, pause, volume adjustment, full-screen mode, etc.).
- Responsive player design for an optimal viewing experience on various devices.

### 3. **Relevant Video Suggestions**

- Displays a curated list of relevant videos based on the currently playing video.
- Enhances user engagement by making content discovery seamless.

### 4. **Search Functionality**

- Fully functional search bar enabling users to find videos using keywords.
- Instant and accurate results, powered by the YouTube API.

### 5. **Modern and Responsive Design**

- A sleek and intuitive user interface designed to enhance usability and aesthetics.
- Fully responsive, providing a seamless experience across desktop, tablet, and mobile devices.

### 6. **API Integration**

- Integration with the YouTube API via RapidAPI for fetching video data, ensuring real-time and up-to-date content availability.

## Tech Stack

### Frontend

- **React.js**: For building an interactive and responsive user interface.
- **CSS & Material-UI**: For styling and ensuring a modern design.

### Backend

- **Node.js & Axios**: For handling API requests.

### Additional Tools

- **RapidAPI**: For YouTube API integration.

## Installation and Setup

### Prerequisites

- Node.js and npm/yarn installed on your machine.

### Steps

1. Clone the repository:

   ```bash
   git clone https://github.com/thaku7469/Vision-Hub.git
   cd visionhub
   ```

2. Install dependencies:

   ```bash
   cd client
   npm install
   ```

3. Configure environment variables:

- Configure environment variables: Create a .env file in the root directory using the env example file:

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open the application in your browser at `http://localhost:3000`.
